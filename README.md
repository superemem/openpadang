# OpenPadang

> Data publik Padang & Sumbar — terbuka, terstandar, gratis.

**Status:** 🌱 Foundation Phase (Mei 2026)

OpenPadang adalah civic tech project independent untuk membuka akses data publik Padang & Sumbar dalam format machine-readable, terstandar, dan gratis. Lahir dari kegelisahan bahwa data pemerintahan saat ini tidak lengkap, terpisah, dan susah diakses.

## Visi

Bantu warga, developer, jurnalis, akademisi akses data Padang/Sumbar tanpa harus scrape ulang setiap kali. Format machine-readable, license terbuka (CC-BY 4.0), API gratis.

## Position

**Komplementer**, bukan kompetitor:
- vs Padang Mobile (Pemkot) — closed system → OpenPadang = open layer publik
- vs Satu Data Indonesia — fokus nasional → OpenPadang = depth lokal Padang/Sumbar
