## 📋 Scope Project

OpenPadang berfokus pada:

- **Data publik Padang & Sumbar** — terutama yang belum memiliki API resmi atau format machine-readable
- **Standardisasi** ke format umum: CSV, JSON, GeoJSON, GTFS
- **Open license:** data CC-BY 4.0, code MIT
- **Komplementer** terhadap layanan resmi pemerintah (Padang Mobile, Satu Data Indonesia)

## 🛠️ Setup Development

**Prerequisite:**

- Node.js ≥20
- pnpm ≥9 (wajib, tidak menggunakan npm/yarn)

```bash
# Clone repository
git clone https://github.com/superemem/openpadang.git
cd openpadang

# Install dependencies
pnpm install

# Jalankan documentation site (Astro Starlight)
pnpm dev:site
# → http://localhost:4321

# Jalankan interactive app (SvelteKit)
pnpm dev:app
# → http://localhost:5173

# Build semua project
pnpm build
```

## 🎨 Standards

### Code Style

- TypeScript strict mode
- Prettier + ESLint (per project)
- **Naming convention:**
  - Folder: `kebab-case`
  - Branch: prefix `feat/`, `fix/`, `docs/`, `chore/`

### Commit Messages

Conventional Commits:

- `feat:` — fitur baru
- `fix:` — perbaikan bug
- `docs:` — perubahan dokumentasi
- `chore:` — maintenance
- `refactor:` — refactor tanpa perubahan behavior

### Dataset Standards

- **Format:** prioritas CSV (RFC 4180), JSON (dengan JSON Schema), GeoJSON, GTFS
- **License:** harus kompatibel dengan CC-BY 4.0
- **Schema:** dokumentasikan fields, types, dan units di README dataset
- **PII:** wajib di-scan dan di-scrub sebelum publish

## 🙏 Code of Conduct

- **Be kind** — diskusi tetap respectful meski terjadi perbedaan pendapat
- **Be patient** — maintainer dan kontributor adalah relawan dengan bandwidth terbatas
- **Be honest** — atribusi yang sesuai dan transparan terhadap limitations
- **Tidak diperkenankan:** spam, harassment, atau diskriminasi

## 💡 Tips untuk Kontributor Baru

- **Tidak perlu menjadi expert** — kontribusi kecil (perbaikan typo, penambahan 1 data source) tetap bernilai
- **Diskusikan via Issue terlebih dahulu** untuk scope besar, agar tidak terjadi tumpang tindih effort
- **Marathon, bukan sprint** — kontribusi 1 dataset per bulan sudah merupakan kontribusi yang signifikan
- **Project bersifat independent** — Anda dipersilakan fork dan melanjutkan versi sendiri apabila diperlukan
