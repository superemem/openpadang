## 🟢 Tier 1: API Real-Time + FREE

Langsung jalan, no partnership. Cocok untuk eksperimen cepat.

### OpenStreetMap (OSM) — Overpass API

<Badge text="Top Priority" variant="success" />

- **Source:** [overpass-turbo.eu](https://overpass-turbo.eu/)
- **Format:** GeoJSON (via Overpass API)
- **License:** ODbL (Open Database License)
- **Refresh:** Real-time community-edited
- **Coverage:** POI, building, road, fasilitas publik Padang

**Use case:** POI Padang (resto, masjid, sekolah, halte, ATM), batas
administratif, road network, building footprints.

### BMKG Gempa Real-Time

- **Source:** [data.bmkg.go.id/DataMKG/TEWS/autogempa.json](https://data.bmkg.go.id/DataMKG/TEWS/autogempa.json)
- **Format:** JSON
- **License:** Public domain (gov data)
- **Refresh:** 6 detik latency
- **Coverage:** Seluruh Indonesia (filter Sumbar di sisi client)

**Other endpoints:**

- `gempaterkini.json` — 15 gempa terakhir
- `gempadirasakan.json` — gempa yang dirasakan warga

### InaTEWS BMKG (Tsunami Early Warning)

- **Source:** [inatews.bmkg.go.id](https://inatews.bmkg.go.id/)
- **Format:** XML / web scrape
- **Refresh:** Real-time
- **Coverage:** Indonesia tsunami warning

**Note:** Critical untuk Padang yang tsunami-prone.

### PetaBencana.id

- **Source:** [petabencana.id](https://petabencana.id)
- **Format:** REST API + GeoJSON
- **License:** Open-source (CC)
- **Refresh:** Real-time crowd-source
- **Coverage:** Multi-bencana, multi-kota Indonesia (filter Sumbar)

### OpenWeatherMap

- **Source:** [openweathermap.org/api](https://openweathermap.org/api)
- **Format:** JSON REST
- **License:** Free tier API (1000 calls/day)
- **Refresh:** 1 jam (free tier)
- **Coverage:** Global, support Padang

### BMKG Cuaca Prakiraan

- **Source:** `api.bmkg.go.id/publik/prakiraan-cuaca?adm4=<kode>`
- **Format:** JSON
- **Refresh:** 6 jam
- **Coverage:** Per kelurahan dengan kode adm4

## 🟡 Tier 2: Stale Tapi Reliable

Perlu scrape, update periodic (tahunan/bulanan).

### BPS Kota Padang

<Badge text="High Priority" variant="caution" />

- **Source:** [padangkota.bps.go.id](https://padangkota.bps.go.id/)
- **Format:** Web table, PDF
- **License:** Government open data
- **Refresh:** Tahunan

**Dataset prioritas:** Penduduk per kecamatan, distribusi umur, sex ratio,
kepadatan, statistik ekonomi.

### BPS Sumatera Barat

- **Source:** [sumbar.bps.go.id](https://sumbar.bps.go.id/)
- **Format:** Web table, PDF
- **Refresh:** Tahunan

### Geoportal Sumbar

- **Source:** [geoportal.sumbarprov.go.id](https://geoportal.sumbarprov.go.id/)
