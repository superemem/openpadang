> **Status:** Catalog sedang disusun. Update terakhir: Mei 2026.

OpenPadang inventarisasi sumber data publik tentang Padang & Sumbar.
Tujuan: **single discovery point** untuk siapa pun yang butuh data — gak
perlu cari ke 10 tempat berbeda.

## Pendekatan: Hybrid

| Tier | Cara Akses | Yang OpenPadang Lakukan |
|---|---|---|
| 🟢 **Direct API** | User akses langsung ke source asli | Bikinin **Quick Start, Code Examples, Use Cases, Gotchas** |
| 🟡 **Curated** | Via OpenPadang API atau download repo | Scrape, transform, standardize, host |
| 🔴 **To Be Curated** | Planned | Roadmap berikutnya |
| 💎 **Partner Required** | Butuh akses resmi | Permohonan ke Dishub/BPBD/dll |

> Untuk Tier "Direct API" yang dokumentasinya jelek atau tidak ada,
> OpenPadang berfungsi sebagai **documentation & education layer**. Setiap
> source punya halaman guide sendiri (planned di Phase 1-2).

## 🟢 Tier 1: Direct API (FREE, langsung jalan)

Sumber-sumber ini **udah ada API resmi & format bagus**. OpenPadang **tidak
re-host** — user akses langsung ke source asli. Yang kami sediakan: dokumentasi
cara pakai & use case Padang-specific.

### OpenStreetMap (OSM) — Overpass API

<Badge text="Guide Coming" variant="caution" />

- **Source:** [overpass-turbo.eu](https://overpass-turbo.eu/)
- **API Endpoint:** `https://overpass-api.de/api/interpreter`
- **Format:** GeoJSON
- **License:** ODbL (Open Database License — attribution wajib)
- **Refresh:** Real-time community-edited
- **Rate Limit:** ~10,000 query/day per IP

**OpenPadang akan sediakan:**
- Quick Start: query Padang POI dalam 5 menit
- Code examples: cURL, JS, Python
- Use cases: cari semua masjid/halte/resto di Padang
- Gotchas: bbox vs area selection, escape characters

### BMKG Gempa Real-Time

<Badge text="Guide Coming" variant="caution" />

- **Source:** [data.bmkg.go.id/DataMKG/TEWS/](https://data.bmkg.go.id/DataMKG/TEWS/)
- **Endpoints:**
  - `autogempa.json` — gempa terbaru (6 detik latency)
  - `gempaterkini.json` — 15 gempa terakhir
  - `gempadirasakan.json` — gempa yang dirasakan warga
- **Format:** JSON
- **License:** Public domain (data pemerintah)
- **Auth:** No API key needed

**OpenPadang akan sediakan:**
- Filter gempa untuk Sumbar (lat/lng range)
- Use cases: alert system, historical analysis, magnitude monitoring
- Gotchas: timezone WIB vs UTC, field naming inconsistency

### BMKG Cuaca Prakiraan

<Badge text="Guide Coming" variant="caution" />

- **Source:** `api.bmkg.go.id/publik/prakiraan-cuaca?adm4=<kode>`
- **Format:** JSON
- **Refresh:** 6 jam
- **Coverage:** Per kelurahan via kode `adm4`

**OpenPadang akan sediakan:**
- Mapping kelurahan Padang → kode adm4
- Use cases: weather widget per kecamatan, daily forecast aggregation

### InaTEWS BMKG (Tsunami Early Warning)

<Badge text="Critical for Padang" variant="danger" />

- **Source:** [inatews.bmkg.go.id](https://inatews.bmkg.go.id/)
- **Format:** XML / scrape
- **Refresh:** Real-time
- **Coverage:** Indonesia tsunami warning

**OpenPadang akan sediakan:**
- Quick start subscribe ke warning feed
- Use cases: tsunami alert PWA, evacuation trigger
- Gotchas: false positive handling, polling vs webhook

### PetaBencana.id

- **Source:** [petabencana.id](https://petabencana.id)
- **API:** [docs.petabencana.id](https://docs.petabencana.id)
- **Format:** REST API + GeoJSON
- **License:** Open-source (CC)
- **Refresh:** Real-time crowd-source

**OpenPadang akan sediakan:**
- Filter Sumbar/Padang province queries
- Use cases: combine dengan BMKG untuk full disaster picture
- Integration patterns dengan map app

### OpenWeatherMap

- **Source:** [openweathermap.org/api](https://openweathermap.org/api)
- **Format:** JSON REST
- **License:** Free tier (1000 calls/day) — butuh API key
- **Coverage:** Global, support Padang

**OpenPadang akan sediakan:**
- Padang-specific city ID setup
- Comparison dengan BMKG (mana lebih akurat untuk kasus apa)
- Use cases: hyperlocal weather widget

## 🟡 Tier 2: Curated by OpenPadang

Sumber yang **format aslinya jelek** atau **gak ada API resmi**. OpenPadang
scrape, transform, standardize, dan host di repo.

### BPS Kota Padang — Demografi

<Badge text="To Curate" variant="note" />

- **Source asli:** [padangkota.bps.go.id](https://padangkota.bps.go.id/) (web table, PDF)
- **OpenPadang akan provide:** CSV time-series (penduduk per kecamatan, age group, sex ratio, kepadatan)
- **Refresh:** Tahunan (manual scrape saat publikasi BPS rilis)

### BPS Sumatera Barat — Ekonomi & Sosial

<Badge text="To Curate" variant="note" />

- **Source asli:** [sumbar.bps.go.id](https://sumbar.bps.go.id/)
- **OpenPadang akan provide:** CSV statistik provinsi (inflasi, kemiskinan, ketenagakerjaan)

### Trans Padang GTFS

<Badge text="To Build" variant="note" />

- **Source asli:** Belum ada GTFS publik (Padang Mobile internal)
- **OpenPadang akan build:** GTFS format community-maintained
- Routes, halte, jadwal — dari article publik + observasi lapangan
- Submit ke MobilityDatabase.org → otomatis muncul di Google Maps Transit
