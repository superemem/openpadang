OpenPadang menginventarisasi sumber data publik tentang Padang dan Sumatera
Barat. Tujuannya menjadi **single discovery point** bagi pengguna yang
membutuhkan data — tanpa perlu mencari di berbagai portal terpisah.

## Pendekatan: Hybrid

| Tier | Cara Akses | Peran OpenPadang |
|---|---|---|
| 🟢 **Direct API** | Akses langsung ke source asli | Menyediakan Quick Start, Code Examples, Use Cases, dan Gotchas |
| 🟡 **Curated** | Melalui API OpenPadang atau download repo | Scrape, transform, standardize, dan host |
| 🔴 **To Be Curated** | Planned | Pada roadmap berikutnya |
| 💎 **Partner Required** | Memerlukan akses resmi | Permohonan kepada Dishub, BPBD, atau institusi terkait |

> Untuk Tier "Direct API" yang dokumentasinya terbatas, OpenPadang berperan
> sebagai **documentation & education layer**. Setiap source memiliki halaman
> guide tersendiri.

---

## 🟢 Tier 1: Direct API (Gratis, Siap Pakai)

Sumber-sumber berikut sudah memiliki API resmi dengan format yang baik.
OpenPadang **tidak melakukan re-hosting** — pengguna mengakses langsung ke
source asli. OpenPadang menyediakan dokumentasi cara penggunaan dan use case
spesifik untuk konteks Padang/Sumbar.

### OpenStreetMap (OSM) — Overpass API

<Badge text="Guide Ready" variant="success" />

- **Source:** [overpass-turbo.eu](https://overpass-turbo.eu/)
- **API Endpoint:** `https://overpass-api.de/api/interpreter`
- **Format:** JSON / GeoJSON
- **License:** ODbL (Open Database License — atribusi wajib)
- **Refresh:** Real-time (community-edited, server snapshot ter-update menit)
- **Rate Limit:** Sekitar 10,000 query per hari per IP

📘 **Dokumentasi OpenPadang:**
- [Quick Start](/guides/osm-overpass/quickstart/) — setup 5 menit, query POI Padang
- [Query Reference](/guides/osm-overpass/endpoints/) — Overpass QL syntax, tag taxonomy, kode wilayah Padang
- [Use Cases Padang](/guides/osm-overpass/use-cases/) — 5 skenario: POI directory, halte map, food finder, emergency locator, tourism
- [Gotchas & Limitations](/guides/osm-overpass/gotchas/) — pitfalls timeout, rate limit, tag inconsistency

### BMKG Gempa Real-Time

<Badge text="Guide Ready" variant="success" />

- **Source:** [data.bmkg.go.id/DataMKG/TEWS/](https://data.bmkg.go.id/DataMKG/TEWS/)
- **Endpoints:**
  - `autogempa.json` — gempa terbaru (latency ~6 detik)
  - `gempaterkini.json` — 15 gempa terakhir
  - `gempadirasakan.json` — gempa yang dirasakan warga
- **Format:** JSON
- **License:** Public domain (data pemerintah)
- **Autentikasi:** Tidak diperlukan

📘 **Dokumentasi OpenPadang:**
- [Quick Start](/guides/bmkg-gempa/quickstart/) — setup 5 menit
- [Endpoint Reference](/guides/bmkg-gempa/endpoints/) — schema lengkap
- [Use Cases Padang](/guides/bmkg-gempa/use-cases/) — 5 skenario implementasi
- [Gotchas & Limitations](/guides/bmkg-gempa/gotchas/) — pitfalls dan quirks

### BMKG Cuaca Prakiraan

<Badge text="Guide Ready" variant="success" />

- **Source:** `api.bmkg.go.id/publik/prakiraan-cuaca?adm4=<kode>`
- **Format:** JSON
- **Refresh:** 6 jam
- **Coverage:** Per kelurahan menggunakan kode `adm4` (granularity hingga 104 kelurahan Padang)

📘 **Dokumentasi OpenPadang:**
- [Quick Start](/guides/bmkg-cuaca/quickstart/) — setup 5 menit
- [Endpoint Reference](/guides/bmkg-cuaca/endpoints/) — schema, weather codes, kode wilayah Padang
- [Use Cases Padang](/guides/bmkg-cuaca/use-cases/) — 5 skenario implementasi
- [Gotchas & Limitations](/guides/bmkg-cuaca/gotchas/) — pitfalls dan quirks

### InaTEWS BMKG (Tsunami Early Warning)

<Badge text="Critical for Padang" variant="danger" />

- **Source:** [inatews.bmkg.go.id](https://inatews.bmkg.go.id/)
- **Format:** XML / web scrape
- **Refresh:** Real-time
- **Coverage:** Indonesia tsunami warning

**Dokumentasi OpenPadang akan menyediakan:**
- Quick start subscribe warning feed
- Use cases: tsunami alert PWA, evacuation trigger
- Gotchas: false positive handling, polling vs webhook

### PetaBencana.id

- **Source:** [petabencana.id](https://petabencana.id)
- **API:** [docs.petabencana.id](https://docs.petabencana.id)
- **Format:** REST API + GeoJSON
- **License:** Open-source (CC)
- **Refresh:** Real-time crowd-source

**Dokumentasi OpenPadang akan menyediakan:**
- Filter Sumbar/Padang province queries
- Use cases: kombinasi dengan BMKG untuk full disaster picture
- Integration patterns dengan map application

### OpenWeatherMap

- **Source:** [openweathermap.org/api](https://openweathermap.org/api)
- **Format:** JSON REST
- **License:** Free tier (1000 calls/day) — memerlukan API key
- **Coverage:** Global, mendukung Padang

**Dokumentasi OpenPadang akan menyediakan:**
- Setup city ID khusus Padang
- Perbandingan dengan BMKG (use case mana yang lebih sesuai)
- Use cases: hyperlocal weather widget

## 🟡 Tier 2: Curated by OpenPadang

Sumber dengan format asli yang sulit dikonsumsi langsung atau belum memiliki
API resmi. OpenPadang melakukan scraping, transformation, standardisasi, dan
hosting di repository.

### BPS Kota Padang — Demografi

<Badge text="To Curate" variant="note" />

- **Source asli:** [padangkota.bps.go.id](https://padangkota.bps.go.id/) (web table, PDF)
- **Output OpenPadang:** CSV time-series (penduduk per kecamatan, age group, sex ratio, kepadatan)
- **Refresh:** Tahunan (manual scrape saat publikasi BPS rilis)

### BPS Sumatera Barat — Ekonomi & Sosial

<Badge text="To Curate" variant="note" />

- **Source asli:** [sumbar.bps.go.id](https://sumbar.bps.go.id/)
- **Output OpenPadang:** CSV statistik provinsi (inflasi, kemiskinan, ketenagakerjaan)

### Trans Padang GTFS

<Badge text="To Build" variant="note" />

- **Source asli:** Belum ada GTFS publik (Padang Mobile bersifat internal)
- **Output OpenPadang:** GTFS format community-maintained
- Routes, halte, jadwal — disusun dari artikel publik dan observasi lapangan
- Submit ke MobilityDatabase.org untuk integrasi otomatis ke Google Maps Transit

## ⚡ Aggregation Endpoints (Planned)

OpenPadang akan menyediakan **combined endpoints** untuk efisiensi konsumsi
data. Satu request menghasilkan response dari multiple sources dengan
filtering Padang-specific.

| Endpoint | Combine | Use Case |
|---|---|---|
| `/api/realtime/padang` | BMKG gempa + BMKG cuaca + PetaBencana banjir + OWM AQI | Dashboard hyperlocal Padang |
| `/api/disaster/sumbar` | BMKG gempa + InaTEWS tsunami + PetaBencana | Disaster operation center |
| `/api/poi/padang` | OSM POI filter Padang + kategorisasi terstandar | Map application |
