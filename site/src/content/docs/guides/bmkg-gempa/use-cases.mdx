## 1. Sistem Alert Real-Time untuk Padang

**Konteks:** Padang berada di wilayah seismik aktif dan tergolong rawan
tsunami. Warga membutuhkan notifikasi cepat ketika terjadi gempa. Notifikasi
resmi terkadang lambat atau tidak mencapai semua pengguna.

**Solusi:** Bot Telegram atau push notification yang mengirim alert untuk
gempa M≥4 di wilayah Sumbar.

### Implementasi

```python
import requests, time

LAST_GEMPA = None

def is_sumbar(wilayah):
    keywords = ['SUMBAR', 'PADANG', 'BUKITTINGGI', 'PARIAMAN', 'MENTAWAI']
    return any(k in wilayah.upper() for k in keywords)

def send_alert(gempa):
    msg = f"⚠️ Gempa M{gempa['Magnitude']} - {gempa['Wilayah']}"
    msg += f"\n📍 {gempa['Coordinates']}"
    msg += f"\n🕐 {gempa['Tanggal']} {gempa['Jam']}"
    msg += f"\n🌊 {gempa['Potensi']}"
    # Kirim melalui Telegram Bot API atau service lain
    print(msg)

while True:
    res = requests.get('https://data.bmkg.go.id/DataMKG/TEWS/autogempa.json')
    gempa = res.json()['Infogempa']['gempa']

    if (
        is_sumbar(gempa['Wilayah'])
        and float(gempa['Magnitude']) >= 4.0
        and gempa['DateTime'] != LAST_GEMPA
    ):
        send_alert(gempa)
        LAST_GEMPA = gempa['DateTime']

    time.sleep(30)
```

**Tech stack yang direkomendasikan:** Python + Telegram Bot API, di-deploy
melalui cron job atau systemd service.

### Implementasi

```ts
// SvelteKit: src/routes/api/gempa-sumbar/+server.ts
import { json } from '@sveltejs/kit';

export async function GET() {
	const res = await fetch('https://data.bmkg.go.id/DataMKG/TEWS/gempaterkini.json');
	const data = await res.json();

	const sumbarGempa = data.Infogempa.gempa.filter((g: any) => {
		const w = g.Wilayah.toUpperCase();
		return ['SUMBAR', 'PADANG', 'MENTAWAI'].some((k) => w.includes(k));
	});

	return json({
		count: sumbarGempa.length,
		latest: sumbarGempa[0] ?? null,
		all: sumbarGempa,
	});
}
```

- Statistics card: total gempa, magnitude rata-rata, kedalaman rata-rata

## 4. Indikator Risiko Tsunami untuk Pesisir Padang

**Solusi:** Indikator yang memperhitungkan magnitude, kedalaman, dan lokasi
(laut vs darat) untuk memberikan level risiko sebagai informasi pelengkap

### Implementasi

```js
function tsunamiRiskLevel(gempa) {
	const mag = parseFloat(gempa.Magnitude);
	const depth = parseInt(gempa.Kedalaman);
	const wilayah = gempa.Wilayah.toLowerCase();

	// Prioritas tertinggi: warning resmi BMKG
	if (gempa.Potensi.includes('berpotensi tsunami')) return 'CRITICAL';

	const isOffShore = wilayah.includes('laut') || wilayah.includes('barat ');

	if (mag >= 6.5 && depth <= 30 && isOffShore) return 'HIGH';
	if (mag >= 5.5 && isOffShore) return 'WATCH';
	return 'NORMAL';
}
```

<Aside type="danger" title="Disclaimer Penting">
	Indikator di atas merupakan **informasi pelengkap berbasis heuristik**,
	BUKAN pengganti peringatan resmi. Warning tsunami resmi hanya dikeluarkan
	oleh BMKG melalui InaTEWS. Prioritaskan pengumuman BMKG/BPBD dalam situasi
	darurat.
</Aside>

### Sudut Cerita yang Dapat Dieksplorasi

- **Frekuensi tahunan** — berapa gempa dirasakan warga Padang dalam setahun?
- **Konsentrasi episentrum** — mengapa Mentawai dominan dalam catatan gempa?
- **Korelasi dengan musim** — adakah pola temporal yang signifikan?
- **Riwayat gempa besar** — kronologi gempa M≥6 di Sumbar 10 tahun terakhir
