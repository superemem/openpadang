## Apa yang Akan Lo Dapet

3 endpoint JSON dari BMKG dengan data gempa Indonesia (termasuk Sumbar):

- **Gempa terbaru** — gempa paling baru (latency ~6 detik)
- **15 gempa terakhir** — riwayat singkat
- **Gempa dirasakan warga** — gempa yang dirasakan masyarakat

Semua **public domain** (data pemerintah), bisa lo pakai untuk **app, dashboard,
riset, atau alert system**.

## Step-by-Step

<Steps>

1. **Buka terminal atau browser**, paste URL ini:

   ```
   https://data.bmkg.go.id/DataMKG/TEWS/autogempa.json
   ```

   Lo bakal dapet response JSON. Itu udah jadi data gempa terbaru.

2. **Parse dalam app lo.** Format response:

   ```json
   {
     "Infogempa": {
       "gempa": {
         "Tanggal": "12 Mei 2026",
         "Jam": "10:15:32 WIB",
         "DateTime": "2026-05-12T03:15:32+00:00",
         "Coordinates": "-0.95,100.36",
         "Lintang": "0.95 LS",
         "Bujur": "100.36 BT",
         "Magnitude": "3.4",
         "Kedalaman": "10 km",
         "Wilayah": "21 km BaratLaut KOTAPADANG-SUMBAR",
         "Potensi": "Tidak berpotensi tsunami",
         "Dirasakan": "II Padang",
         "Shakemap": "20260512031532.mmi.jpg"
       }
     }
   }
   ```

3. **Filter Sumbar/Padang.** Cek `Wilayah` mengandung kata `SUMBAR`, `PADANG`,
   atau `BUKITTINGGI`, atau parse koordinat (Sumbar ada di **lat -3.5 s/d 0.5,
   lng 98 s/d 102**).

4. **Done!** Lo bisa display di app, send notification, atau analisa.

</Steps>

## Filter Padang/Sumbar

Karena BMKG endpoint cover seluruh Indonesia, lo perlu filter di sisi client:

```js
function isPadangSumbar(gempa) {
	const wilayah = gempa.Wilayah.toUpperCase();
	if (wilayah.includes('SUMBAR') || wilayah.includes('PADANG')) return true;
	if (wilayah.includes('BUKITTINGGI') || wilayah.includes('PARIAMAN')) return true;

	// Atau cek koordinat (bbox Sumbar approx)
	const [lat, lng] = gempa.Coordinates.split(',').map(Number);
	return lat >= -3.5 && lat <= 0.5 && lng >= 98 && lng <= 102;
}
```
