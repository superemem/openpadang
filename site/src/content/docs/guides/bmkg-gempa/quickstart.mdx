<Aside type="tip" title="Setup dalam 5 menit">
	BMKG menyediakan endpoint publik tanpa autentikasi. Cukup HTTP GET ke URL
	JSON dan Anda siap mengonsumsi data gempa real-time.

## Apa yang Akan Diperoleh

API BMKG menyediakan 3 endpoint JSON untuk data gempa Indonesia (termasuk
wilayah Sumatera Barat):

- **Gempa terbaru** — gempa paling baru dengan latency sekitar 6 detik
- **15 gempa terakhir** — riwayat gempa untuk dashboard atau analisis
- **Gempa dirasakan warga** — gempa yang dilaporkan dirasakan masyarakat

Seluruh data merupakan **public domain** (data pemerintah) dan dapat digunakan
untuk berbagai keperluan: aplikasi, dashboard, sistem alert, riset akademik,
maupun integrasi internal.

## Langkah-Langkah

<Steps>

1. **Akses URL endpoint** menggunakan terminal atau browser:

   ```
   https://data.bmkg.go.id/DataMKG/TEWS/autogempa.json
   ```

   Response berupa data gempa terbaru dalam format JSON.

2. **Parse response** dalam aplikasi Anda. Struktur response:

   ```json
   {
     "Infogempa": {
       "gempa": {
         "Tanggal": "12 Mei 2026",
         "Jam": "10:15:32 WIB",
         "DateTime": "2026-05-12T03:15:32+00:00",
         "Coordinates": "-0.95,100.36",
         "Lintang": "0.95 LS",
         "Bujur": "100.36 BT",
         "Magnitude": "3.4",
         "Kedalaman": "10 km",
         "Wilayah": "21 km BaratLaut KOTAPADANG-SUMBAR",
         "Potensi": "Tidak berpotensi tsunami",
         "Dirasakan": "II Padang",
         "Shakemap": "20260512031532.mmi.jpg"
       }
     }
   }
   ```

3. **Filter berdasarkan wilayah** untuk fokus pada Sumbar/Padang. Gunakan
   field `Wilayah` (cek keyword `SUMBAR`, `PADANG`, `BUKITTINGGI`, dst) atau
   parse koordinat (Sumbar berada pada **lat -3.5 hingga 0.5, lng 98 hingga 102**).

4. **Implementasikan** sesuai kebutuhan: tampilkan di antarmuka, kirim
   notifikasi, atau lakukan analisis lebih lanjut.

</Steps>

```js
const res = await fetch('https://data.bmkg.go.id/DataMKG/TEWS/autogempa.json');
```

## Filter Wilayah Padang/Sumbar

Endpoint BMKG menyediakan data untuk seluruh Indonesia. Untuk fokus pada
wilayah Padang/Sumbar, lakukan filtering pada sisi client:

```js
function isPadangSumbar(gempa) {
	const wilayah = gempa.Wilayah.toUpperCase();
	if (wilayah.includes('SUMBAR') || wilayah.includes('PADANG')) return true;
	if (wilayah.includes('BUKITTINGGI') || wilayah.includes('PARIAMAN')) return true;

	// Alternatif: bounding box Sumatera Barat
	const [lat, lng] = gempa.Coordinates.split(',').map(Number);
	return lat >= -3.5 && lat <= 0.5 && lng >= 98 && lng <= 102;
}
```
